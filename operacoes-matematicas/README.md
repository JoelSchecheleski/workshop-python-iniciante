# Operações matemáticas

Operadores numéricos básicos:
- Adição: +
- Multiplicação: *
- Subtração: -
- Potenciação: **
- Divisão: /
- Resto de uma divisão: %

Exemplos:
```python
a = 2
b = 3
a + b

a * b # multiplicação

10 % 8 # resto da divisão entre números inteiros

5 / 2 # quero que retorne um número "quebrado"

5 // 2 # quero que retorne um inteiro

(a + b - 2 * a + a ** a) / 5
```
